# Proxy Toggle Script

Скрипт для Debian, который позволяет интерактивно:

- включить proxy
- отключить proxy

## Что настраивается

При включении proxy скрипт создает и настраивает:

- `/etc/profile.d/99-system-proxy.sh` — системные переменные окружения
- `/etc/apt/apt.conf.d/95proxy` — proxy для `apt`
- `/etc/sudoers.d/proxy-env` — сохранение proxy-переменных при использовании `sudo`

При отключении proxy эти файлы удаляются.

## Требования

- Debian
- запуск от `root`

## Запуск

```bash
chmod +x proxy-toggle.sh
sudo ./proxy-toggle.sh
```

## Режимы работы

После запуска скрипт предлагает:

1. Включить proxy
2. Отключить proxy
3. Выйти

## Что запрашивается при включении

* схема proxy (`http` или `https`)
* адрес proxy
* порт
* логин и пароль, если нужна авторизация
* список `NO_PROXY`

## Важно

* настройки `apt` начинают работать сразу
* для текущей shell-сессии после включения можно выполнить:

```bash
source /etc/profile.d/99-system-proxy.sh
```

* после отключения в текущей shell-сессии можно очистить переменные вручную:

```bash
unset http_proxy https_proxy HTTP_PROXY HTTPS_PROXY no_proxy NO_PROXY
```

* для уже запущенных сервисов может потребоваться перезапуск

## Назначение

Скрипт предназначен для настройки **исходящего proxy** (outbound proxy), через который машина выходит во внешнюю сеть.
Он не предназначен для настройки reverse proxy (`nginx`, `haproxy` и т.д.).
